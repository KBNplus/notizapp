# NotizApp

Eine kleine Progressive Web App zum Verwalten von Notizen. Beim Start erscheint kurz ein Splash-Screen mit Name und Version, danach kann man Notizen erstellen, verschieben, als erledigt markieren und wieder löschen. Die Daten werden lokal im Browser gespeichert.

## Funktionen
- Splash-Screen mit Name und Version
- Notizen anlegen
- Notizen als erledigt markieren
- Notizen verschieben (hoch/runter)
- Notizen löschen
- Offline nutzbar durch Service Worker

## Nutzung
Die Anwendung besteht ausschließlich aus statischen Dateien. Einfach `index.html` im Browser öffnen oder sie auf einen Webserver legen. Beim ersten Aufruf können die Dateien installierbar gemacht werden, so dass die App wie eine native Anwendung genutzt werden kann.
